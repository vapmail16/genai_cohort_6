# Neural Network: Next Word Prediction Demo

An interactive Streamlit presentation that explains how neural networks predict the next word in a sequence.

## Example Sentence
"a cat sat on a" → predicts next word

## Features
- Step-by-step explanation of neural network concepts
- High-level understanding (not deep mathematical details)
- Interactive presentation format with navigation
- Visual demonstrations

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running the App

```bash
streamlit run app.py
```

The app will open in your browser automatically.

## Navigation
- Use **Next ▶** and **◀ Previous** buttons to navigate through slides
- Total of 8 slides covering:
  1. Introduction
  2. The Problem
  3. Word Embeddings
  4. Neural Network Architecture
  5. Learning Patterns
  6. Processing the Example
  7. Making Predictions
  8. Key Takeaways
